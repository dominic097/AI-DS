Excersise 
Image-Based Plant Disease Detection

Scenario:
Farmers need an automated system to detect plant diseases using leaf images.

Mini Project Task:
Build a system that classifies plant leaves into healthy or diseased categories.

Expected Techniques (Units 1–2):
- Image preprocessing
- Color and texture feature extraction
- Classification (CNN / SVM)
- Dataset training

Student Deliverables:
- Disease detection model
- Accuracy and loss graphs
- Sample predictions
- Report with dataset details
