

# Linear (Identity) Activation Function
The linear activation function, also known as the identity activation function, is a simple function that outputs the input directly without any transformation. It is defined mathematically as:

```math
f(x) = x
```

Sigmoid Activation Function
The sigmoid activation function is a commonly used activation function in neural networks, especially for binary classification tasks

It maps any real-valued number into the range (0, 1). The sigmoid function is defined mathematically as:

```math
f(x) = \frac{1}{1 + e^{-x}}
```

Where:
- `e` is the base of the natural logarithm (approximately equal to 2.71828)
- `x` is the input to the function  

The sigmoid function has an S-shaped curve and is characterized by the following properties:
- Output Range: The output values of the sigmoid function range between 0 and 1. This makes it suitable for models where we need to predict probabilities.
- Smooth Gradient: The sigmoid function has a smooth gradient, which helps in optimizing the weights during backpropagation.
- Non-linearity: The sigmoid function introduces non-linearity into the model, allowing it to learn complex patterns in the data.
- Derivative: The derivative of the sigmoid function can be expressed in terms of the function itself:
```math
f'(x) = f(x) \cdot (1 - f(x))
```


# Tanh Activation - Balanced Signal Modeling: 

The hyperbolic tangent (tanh) activation function is another commonly used activation function in neural networks. It maps any real-valued number into the range (-1, 1). The tanh function is defined mathematically as:

```math
f(x) = \frac{e^{x} - e^{-x}}{e^{x} + e^{-x}}
```

Where:
- `e` is the base of the natural logarithm (approximately equal to 2.71828)
- `x` is the input to the function

The tanh function has an S-shaped curve similar to the sigmoid function but is centered around zero. It is characterized by the following properties:
- Output Range: The output values of the tanh function range between -1 and 1. This makes it suitable for models where the output can have both positive and negative values.
- Smooth Gradient: The tanh function has a smooth gradient, which helps in optimizing the weights during backpropagation.
- Non-linearity: The tanh function introduces non-linearity into the model, allowing it to learn complex patterns in the data.
- Derivative: The derivative of the tanh function can be expressed in terms of the function itself:
```math
f'(x) = 1 - f(x)^2
``` 


# ReLu Activation - Sparse Activation and Efficient Computation:
The Rectified Linear Unit (ReLU) activation function is widely used in deep learning due to its simplicity and effectiveness. It is defined mathematically as:

```math
f(x) = \max(0, x)
```

# Leaky Relu:
The Leaky Rectified Linear Unit (Leaky ReLU) activation function is a variant of the standard ReLU activation function that allows a small, non-zero gradient when the input is negative. This helps to address the "dying ReLU" problem, where neurons can become inactive and only output zero for all inputs. The Leaky ReLU function is defined mathematically as:

```math
f(x) = \begin{cases} x & \text{if } x > 0 \\ \alpha x & \text{if } x \leq 0 \end{cases}
```
Where:
- `x` is the input to the function
- `α` (alpha) is a small constant (typically 0.01) that determines the slope of the function for negative input values

# softmax Activation function: - Probabilistic Multi-class Classification:
The softmax activation function is commonly used in multi-class classification problems, particularly in the output layer of neural networks. It converts a vector of raw scores (logits) into probabilities, where each probability represents the likelihood of each class. The softmax function is defined mathematically as:    

```math
f(x_i) = \frac{e^{x_i}}{\sum_{j=1}^{K} e^{x_j}}
```
Where:
- `x_i` is the input score (logit) for class `i`
- `K` is the total number of classes
- The output `f(x_i)` is the probability of class `i`, and the sum of all output probabilities equals 1.

# Feedforward Neural network:

A feedforward neural network is a type of artificial neural network where the connections between the nodes do not form a cycle. In this architecture, information moves in only one direction—from the input layer, through the hidden layers, and finally to the output layer. There are no loops or cycles in the network.

Type of Feedforward Neural Networks:
1. Single-Layer Perceptron: This is the simplest form of a feedforward neural network, consisting of a single layer of output nodes connected directly to the input nodes. It can   only solve linearly separable problems.
2. Multi-Layer Perceptron (MLP): This type of feedforward neural network contains one or more hidden layers between the input and output layers. MLPs can model complex relationships and solve non-linear problems.


# BPN - Back Propagation Neural Network
Backpropagation (backward propagation of errors) is a supervised learning algorithm used for training artificial neural networks. It is a method for calculating the gradient of the loss function with respect to the weights of the network, allowing for efficient weight updates during training.

Uses gradient descent to minimize the error by adjusting the weights in the network based on the error signal propagated backward from the output layer to the input layer.

### Backpropagation Algorithm Steps:
1. **Initialization**: Initialize the weights and biases of the neural network randomly or using a specific initialization method.
2. **Forward Pass**: For each training example, perform a forward pass through the network to compute the output predictions. This involves calculating the activations of each neuron layer by layer, starting from the input layer to the output layer.
3. **Compute Loss**: Calculate the loss (error) between the predicted output and the actual target values using a suitable loss function (e.g., Mean Squared Error, Cross-Entropy Loss).
4. **Backward Pass**: Perform a backward pass through the network to compute the gradients of the loss with respect to each weight and bias in the network. This is done using the chain rule of calculus to propagate the error backward from the output layer to the input layer.
mathematically, for each weight `w_ij` connecting neuron `i` to neuron `j`, the gradient is computed as:
   
   ```math
   \frac{\partial L}{\partial w_{ij}} = \delta_j \cdot a_i
   ```
   Where:
   - `L` is the loss function
   - `δ_j` is the error term for neuron `j`
   - `a_i` is the activation of neuron `i`

5. **Weight Update**: Update the weights and biases using the computed gradients and a learning rate. The weights are adjusted in the direction that reduces the loss.
mathematically, the weight update can be expressed as:
   
   ```math
   w_{ij} = w_{ij} - \eta \cdot \frac{\partial L}{\partial w_{ij}}
   ```
   Where:
   - `η` (eta) is the learning rate
   - `∂L/∂w_ij` is the gradient of the loss with respect to the weight `w_ij`

6. **Repeat**: Repeat the forward pass, loss computation, backward pass, and weight update steps for a specified number of epochs or until convergence (when the loss stops decreasing significantly).

### Example Code for Backpropagation in a Simple Neural Network

```python
import numpy as np
# Sigmoid activation function
def sigmoid(x):
    return 1 / (1 + np.exp(-x)) 

def sigmoid_derivative(x):
    return x * (1 - x)

# Simple Neural Network class
class SimpleNN:
    def __init__(self, input_size, hidden_size, output_size, learning_rate=0.1):
        self.learning_rate = learning_rate
        self.weights_input_hidden = np.random.rand(input_size, hidden_size)
        self.weights_hidden_output = np.random.rand(hidden_size, output_size)
        self.bias_hidden = np.random.rand(hidden_size)
        self.bias_output = np.random.rand(output_size)
    def forward(self, X):
        self.hidden_input = np.dot(X, self.weights_input_hidden) + self.bias_hidden
        self.hidden_output = sigmoid(self.hidden_input)
        self.final_input = np.dot(self.hidden_output, self.weights_hidden_output) + self.bias_output
        self.final_output = sigmoid(self.final_input)
        return self.final_output
    def backward(self, X, y):
        output_error = y - self.final_output
        output_delta = output_error * sigmoid_derivative(self.final_output)
        hidden_error = output_delta.dot(self.weights_hidden_output.T)
        hidden_delta = hidden_error * sigmoid_derivative(self.hidden_output)
        self.weights_hidden_output += self.hidden_output.T.dot(output_delta) * self.learning_rate
        self.weights_input_hidden += X.T.dot(hidden_delta) * self.learning_rate
        self.bias_output += np.sum(output_delta, axis=0) * self.learning_rate
        self.bias_hidden += np.sum(hidden_delta, axis=0) * self.learning_rate   
    def train(self, X, y, epochs):
        for epoch in range(epochs):
            self.forward(X)
            self.backward(X, y)
# Example usage
if __name__ == "__main__":
    # Input data (X) and target output (y)
    X = np.array([[0, 0], [0, 1], [1, 0], [1, 1]])
    y = np.array([[0], [1], [1], [0]])  # XOR problem
    # Create and train the neural network
    nn = SimpleNN(input_size=2, hidden_size=2, output_size=1, learning_rate=0.5)
    nn.train(X, y, epochs=10000)
    # Test the neural network
    print("Final outputs after training:")
    print(nn.forward(X))    

```
Assume W =5 learninf rate = 0.1 Intiial G = 0   
Iteration 1:
- Compute prediction: Y_pred = X * W = [5, 10, 15, 20]
- Calculate loss: Loss = Mean((Y - Y_pred)^2) = Mean(([2, 4, 6, 8] - [5, 10, 15, 20])^2) = Mean(([ -3, -6, -9, -12])^2) = Mean([9, 36, 81, 144]) = 67.5
- Compute gradients:
  - dL/dW = (-2/N) * Σ(X * (Y - Y_pred)) = (-2/4) * (1*(-3) + 2*(-6) + 3*(-9) + 4*(-12)) = -0.5 * (-3 - 12 - 27 - 48) = -0.5 * (-90) = 45
- Update G: G = 0 + (45)^2 = 2025
- Update weight: W = 5 - (0.1 / (sqrt(2025) + 1e-8)) * 45 = 5 - (0.1 / (45 + 1e-8)) * 45 = 5 - 0.1 = 4.9
Iteration 2:
- Compute prediction: Y_pred = X * W = [4.9, 9.8, 14.7, 19.6]
- Calculate loss: Loss = Mean((Y - Y_pred)^2) = Mean(([2, 4, 6, 8] - [4.9, 9.8, 14.7, 19.6])^2) = Mean(([-2.9, -5.8, -8.7, -11.6])^2) = Mean([8.41, 33.64, 75.69, 134.56]) = 63.075
- Compute gradients:
  - dL/dW = (-2/N) * Σ(X * (Y - Y_pred)) = (-2/4) * (1*(-2.9) + 2*(-5.8) + 3*(-8.7) + 4*(-11.6)) = -0.5 * (-2.9 - 11.6 - 26.1 - 46.4) = -0.5 * (-87) = 43.5
- Update G: G = 2025 + (43.5)^2 = 3922.25
- Update weight: W = 4.9 - (0.1 / (sqrt(3922.25) + 1e-8)) * 43.5 = 4.9 - (0.1 / (62.6 + 1e-8)) * 43.5 = 4.9 - 0.0695 = 4.8305
### Note:
- The above example demonstrates the backpropagation algorithm in a simple neural network with one hidden layer. In practice, neural networks can have multiple hidden layers and more complex architectures.
- The learning rate, number of epochs, and network architecture can significantly impact the training process and the final performance of the model.
### Where:
- `L` is the loss function
- `E[g^2]` is the moving average of the squared gradients
- `dL/dweight` and `dL/dbias` are the gradients of the loss with respect to the weights and biases
- `epsilon` is a small constant added for numerical stability
