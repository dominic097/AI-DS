# Adagrad
Adagrad (Adaptive Gradient Algorithm) is an optimization algorithm designed to adapt the learning rate for each parameter individually based on the historical gradients. It is particularly useful for dealing with sparse data and features.

The key idea behind Adagrad is to maintain a separate learning rate for each parameter, which decreases over time as more updates are made. This allows the algorithm to perform larger updates for infrequent parameters and smaller updates for frequent parameters.

### Mathematical Formula: 


### Adagrad Update Rule:
The update rule for Adagrad is as follows:
```weight = weight - (learning_rate / (sqrt(G) + epsilon)) * dL/dweight
bias = bias - (learning_rate / (sqrt(G) + epsilon)) * dL/dbias
```
where:
- `G` is the sum of the squares of the gradients for each parameter up to the current time step.
- `epsilon` is a small constant added to prevent division by zero.

### Algorithm Steps:
1. Initialize weights and biases randomly.
2. Initialize `G` (the sum of squared gradients) to zero for each parameter.
3. For each iteration:
    - Compute the predictions for the entire dataset.
    - Calculate the loss using the loss function.
    - Compute the gradients of the loss with respect to weights and biases.
    - Update `G` by adding the square of the current gradients.
    - Update weights and biases using the Adagrad update rules.
4. Repeat until convergence or for a fixed number of iterations.

### Example Code for Adagrad

```python
import numpy as np
X = np.array([1, 2, 3, 4])
Y = np.array([2, 4, 6, 8])

w = 0
lr = 0.1
G = 0
epsilon = 1e-8

# Training loop: iterate for 1000 epochs
for epoch in range(1000):
    Y_pred = X * w
    
    # Calculate loss and gradients
    loss = np.mean((Y - Y_pred) ** 2)
    
    # Compute gradients
    dw = (-2 / len(X)) * np.sum(X * (Y - Y_pred))
    
    # Update G
    G += dw ** 2
    
    # Update weight using Adagrad rule
    w = w - (lr / (np.sqrt(G) + epsilon)) * dw
    
    if epoch % 100 == 0:
        print(f'Epoch {epoch}, Loss: {loss}, Weight: {w}')
```

Assume W =5 learninf rate = 0.1 Intiial G = 0

Iteration 1:
- Compute prediction: Y_pred = X * W = [5, 10, 15, 20]
- Calculate loss: loss = mean((Y - Y_pred)^2) = mean(([2, 4, 6, 8] - [5, 10, 15, 20])^2) = 54.75
- Compute gradients: dw = (-2 / 4) * sum(X * (Y - Y_pred)) = -110
- Update G: G = 0 + (-110)^2 = 12100
- Update weight: W = 5 - (0.1 / (sqrt(12100) + 1e-8)) * -110 = 5.099

Iteration 2:
- Compute prediction: Y_pred = X * W = [5.099, 10.198, 15.297, 20.396]
- Calculate loss: loss = mean((Y - Y_pred)^2) = mean(([2, 4, 6, 8] - [5.099, 10.198, 15.297, 20.396])^2) = 48.802
- Compute gradients: dw = (-2 / 4) * sum(X * (Y - Y_pred)) = -99.8
- Update G: G = 12100 + (-99.8)^2 = 22160.
- Update weight: W = 5.099 - (0.1 / (sqrt(22160) + 1e-8)) * -99.8 = 5.164   


# RMSProp: 
RMSProp (Root Mean Square Propagation) is an adaptive learning rate optimization algorithm that addresses the diminishing learning rates problem of Adagrad. It does this by maintaining a moving average of the squared gradients, which allows it to adapt the learning rate for each parameter based on recent gradient information.

### RMSProp Update Rule:
The update rule for RMSProp is as follows:```weight = weight - (learning_rate / (sqrt(E[g^2]) + epsilon)) * dL/dweight
bias = bias - (learning_rate / (sqrt(E[g^2]) + epsilon)) * dL/dbias

```where:
- `E[g^2]` is the moving average of the squared gradients for each parameter.
- `epsilon` is a small constant added to prevent division by zero.


### Mathematical Formula:

**Step 1: Compute Gradient**
```
g_t = ∂L/∂θ_t
```
Calculate the gradient of the loss function with respect to the parameters at time step t.

**Step 2: Update Moving Average of Squared Gradients (Second Moment)**
```
E[g^2]_t = β * E[g^2]_(t-1) + (1 - β) * g_t^2
```
Where:
- `β` is the decay rate (typically 0.9 or 0.99)
- `E[g^2]_t` is the exponential moving average of squared gradients
- This step implements the RMSProp idea: use a moving average instead of accumulating all gradients

**Step 3: Compute Parameter Update**
```
θ_(t+1) = θ_t - (η / sqrt(E[g^2]_t + ε)) * g_t
```
Where:
- `η` is the learning rate
- `ε` is a small constant (typically 1e-8) to prevent division by zero
- The learning rate is adapted based on the moving average of squared gradients

**Key Differences from Adagrad:**
- RMSProp uses exponential moving average of squared gradients instead of cumulative sum
- This prevents the learning rate from becoming infinitesimally small
- The decay parameter β controls how much history to consider


### Algorithm Steps:

1. Initialize weights and biases randomly.
2. Initialize `E[g^2]` (the moving average of squared gradients) to zero for each parameter.
3. For each iteration:
    - Compute the predictions for the entire dataset.
    - Calculate the loss using the loss function.
    - Compute the gradients of the loss with respect to weights and biases.
    - Update `E[g^2]` using the current gradients.
    - Update weights and biases using the RMSProp update rules.
4. Repeat until convergence or for a fixed number of iterations.

### Example Code for RMSProp

```python
import numpy as np
X = np.array([1, 2, 3, 4]) 
Y = np.array([2, 4, 6, 8])
w = 0
lr = 0.01
E_g2 = 0
beta = 0.9
epsilon = 1e-8

# Training loop: iterate for 1000 epochs
for epoch in range(1000):
    Y_pred = X * w
    
    # Calculate loss and gradients
    loss = np.mean((Y - Y_pred) ** 2)
    
    # Compute gradients
    dw = (-2 / len(X)) * np.sum(X * (Y - Y_pred))
    
    # Update E[g^2]
    E_g2 = beta * E_g2 + (1 - beta) * dw ** 2
    
    # Update weight using RMSProp rule
    w = w - (lr / (np.sqrt(E_g2) + epsilon)) * dw
    
    if epoch % 100 == 0:
        print(f'Epoch {epoch}, Loss: {loss}, Weight: {w}')
```
### Example Scenarios
1) Computer Vision: RMSProp is widely used in training convolutional neural networks (CNNs) for image classification, object detection, and image segmentation tasks.
2) Natural Language Processing: In NLP tasks like sentiment analysis or machine translation, RMSProp is used to train deep learning models such as recurrent neural networks (RNNs) and transformers.
3) Reinforcement Learning: RMSProp is often employed in reinforcement learning algorithms, such as Deep Q-Networks (DQN), to optimize the policy and value functions.
4) Speech Recognition: RMSProp can be applied to optimize models for automatic speech recognition (ASR) systems, improving their accuracy and performance.
