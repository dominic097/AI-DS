## Key Takeaways

1. **Data Structure** determines preprocessing needs and algorithm choices
2. **Problem Type** (classification vs regression) depends on output requirements
3. **Algorithm Selection** should consider interpretability, dataset size, and feature relationships  
4. **Error Types** (FP vs FN) have different business costs and should guide model optimization
5. **Positive Class** should be the rare/important outcome you're trying to detect

## Quick Reference

- **Classification**: Discrete categories → "Will it rain?" (Yes/No)
- **Regression**: Continuous numbers → "How many inches of rain?" (2.3 inches)
- **True = Correct prediction**, **False = Wrong prediction**
- **Positive = Expected result**, **Negative = Normal/absence**
- **FP = False Alarm**, **FN = Missed Detection**
