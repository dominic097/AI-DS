
A point estimate is a single best-value estimate of an unknown population parameter (for example the population mean μ, proportion p, variance σ², or a regression coefficient β) computed from sample data.

What it is

- A single number (no uncertainty attached) used as your best guess for the true parameter.
- Example: the sample mean x̄ is a point estimate of μ; the sample proportion p̂ is a point estimate of p.

Common point estimators and formulas

- Sample mean: x̄ = (Σ xi) / n
- Sample proportion: p̂ = x / n (x = number of successes)
- Sample variance (unbiased estimator of population variance): s² = Σ(xi − x̄)² / (n − 1)
- Difference of means: (x̄1 − x̄2)
- Difference of proportions: (p̂1 − p̂2)

Demerits of Point Estimates 
* the point estimate is not accurate in most of the cases 
* it is single value representation of the population that differs with each sample 
* the deviation of the point estimate from the population parameter gives rise to sampling error 

## Interval Estimate

An interval estimate gives a range of plausible values for an unknown population parameter (e.g., mean μ, proportion p) together with a confidence level that quantifies how the interval procedure performs in repeated sampling.

It is a range or an interval within which the parameter lies for a stated confidence level 

What it is

- A confidence interval (CI) is written as: point estimate ± margin of error (or as [lower, upper]).
- Common notation: 95% CI for μ = x̄ ± t* · SE(x̄).
- The “95%” refers to the method: if you repeated the sampling many times and computed CIs the same way, ≈95% of them would contain the true parameter.
